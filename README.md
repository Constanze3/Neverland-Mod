# Neverland

Dependencies:
- Fabric API
- Origins
- Pehkui

A collection of origins created originally for a server I hosted.
The goal of the origins is to make playing minecraft more fun in creative ways.

Origins:
- Harbinger
- Jester
- Nature Golem
- Storm Spirit
- Werewolf

Traits:
- Auto Cooker
- Bunny Hopping
- Concrete Converter
- Diamon Maniac

Let me know if you find any major issues/bugs.
